Sobre mim

Programação é o processo de criação de um conjunto de instruções que dizem ao computador como realizar uma tarefa. Essas instruções são escritas em um idioma do computador, como SQL, Java, Python e C++. O programador é o profissional que escreve, verifica, averigua e mantém o código-fonte, que é o texto que contém as instruções.
